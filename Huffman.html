(a) Develop a Huffman code for the given pmf px(x), calculate its average codeword length and its absolute and relative redundancy.

Solution:
The Huffman algorithm can be described as follows: First, we create a symbol group for each alphabet letter. Then, in each iteration, the symbol groups are sorted according to their associated probabilities. Two symbol groups with the smallest probabilities are selected, and each of the two symbol groups is characterized by a single bit. Then, the two selected symbol groups are summarized to a new symbol group. This process is repeated until a single symbol group is ob- tained. Finally, the constructed binary code tree is converted into a prefix code using the assigned bits. The construction of the binary code tree for the given pmf is illus- trated in the following table.
